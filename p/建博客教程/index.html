建博客教程

我的第一篇博客，有点粗糙

Aug 01, 2021
1 min read

安装hugo

用hugo version 看安装是否成功

用hugo建博客

访问这个网址下载对应喜欢的主题

hugo new site （站点名） –创建博客站点（初始化博客目录 ）

git clone （themes资源） –将资源放到站点内（注意要在站点根目录下用）

hugo new post/（博客名.md）–创建自己的博客（注意draft: true要去掉）

预览自己博客

hugo server –访问127.0.0.1:1313端口预览

#发布自己博客 hugo –baseUrl=“仓库地址” –buildDrafts –会生成public文件夹，同时站点监听域名（buildDrafts表示包括被标记为draft的文章；）

hugo –生成并更新public（之后可这样）

cd public –进入public

git init –第一次要在这个文件夹下初始化git

git add . –提交所有文件到暂存区

git status –可看提交的内容是否成功

git commit -m “我的hugo博客第一次提交” –提示内容且放到本地仓库

注意创建时要/自己github号.github.io

git remote add origin https://github.com/（自己git号）/(仓库地址).git –第一次要将本地仓库与远程仓库建立联系

这里出错了可以先 git remote rm origin –之前有建可删除在重新绑定发布地址

git push -u origin master –输入自己账号密码，提交(第一次要)

git push origin master –之后可这样

git config –global –unset http.proxy –绿色上网后要关代理在试
